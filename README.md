# Code Rouge - Escape Game Securite Incendie

Serious game collaboratif sur tablette pour la formation EPI.

## Deploy sur Vercel

1. Push ce dossier sur un repo GitHub
2. Va sur vercel.com, connecte ton GitHub
3. New Project > selectionne le repo > Framework: Vite > Deploy

## Dev local

npm install && npm run dev

## Stack

React 18 + Vite + Tone.js + Canvas particles + SVG characters
